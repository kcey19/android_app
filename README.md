## Currency rate application
Practiced developing a modern multi screen Android application using Kotlin and Jetpack Compose.
Integrated exchange rate API to display currency rates with respect to USD$. 
Played around with UI layouts, implemented state management and navigation.

 Modern UI design 
 State Management
 Search and Navigation
